### 1. 获取 Baidu Analytics (百度统计) `ba_track_id`

百度统计的 `ba_track_id` 通常是一个 32 位的长字符串（哈希值）。

**操作步骤：**

1. **注册/登录：** 访问 [百度统计官网 (tongji.baidu.com)](https://tongji.baidu.com/) 并使用百度账号登录。
2. **添加网站：**
* 进入“管理”页面（通常在顶部导航栏）。
* 在“网站列表”标签页下，点击 **“新增网站”**。
* 输入你的网站域名（例如 `yourblog.com`）和网站首页地址，点击确定。


3. **获取代码/ID：**
* 添加成功后，系统会跳转到“代码获取”页面。
* 你会看到一段 JavaScript 代码。你需要提取其中 `hm.src` 链接里 `?` 后面的一长串字符。
* **代码示例：**
```javascript
hm.src = "https://hm.baidu.com/hm.js?83e259f69b37d02a4633a2b7d960139c";

```


* **你的 ID：** 上面代码中 `?` 后的 `83e259f69b37d02a4633a2b7d960139c` 就是你需要填入配置文件 `ba_track_id` 处的内容。



---

### 2. 获取 Google Analytics (谷歌分析) `ga_track_id`

**重要提示：**
你配置中的 `UA-90855596-1` 是旧版 **Universal Analytics (UA)** 的格式。Google 已于 **2023年7月1日** 停止处理 UA 数据。现在新创建的媒体资源都是 **Google Analytics 4 (GA4)**，格式为 `G-XXXXXXXXXX`。

如果你的博客主题较旧，可能只支持 `UA-` 格式。建议检查主题文档是否支持 GA4，或者尝试直接填入 GA4 的 ID 测试是否生效。

**操作步骤 (获取 GA4 ID)：**

1. **注册/登录：** 访问 [Google Analytics (analytics.google.com)](https://www.google.com/search?q=https://analytics.google.com/) 并登录 Google 账号。
2. **创建账号/媒体资源：**
* 点击左下角的 **齿轮图标 (管理)**。
* 点击 **“创建账号”**（如果是第一次使用）或 **“创建媒体资源”**。
* 输入属性名称（如“我的博客”），设置时区和货币。


3. **设置数据流：**
* 在平台选择页面，选择 **“网站 (Web)”**。
* 输入你的网站网址（如 `www.yourblog.com`）和流名称。
* 点击 **“创建数据流”**。


4. **获取衡量 ID (Measurement ID)：**
* 创建完成后，你会看到“网站数据流详情”页面。
* 复制右上角的 **“衡量 ID”**（格式为 `G-` 开头，例如 `G-A1B2C3D4E5`）。
* 将此 ID 填入配置文件的 `ga_track_id` 处。



> **如果是旧主题不兼容 GA4 怎么办？**
> 很多旧的 Hexo/Jekyll 主题模板中的 Google Analytics 代码块是写死的旧版逻辑（寻找 `UA-` 前缀）。如果填入 `G-` 开头的 ID 不生效，你需要手动修改主题的 `.ejs` 或 `.swig` 模板文件，将旧的 `gtag.js` 代码替换为 Google Analytics 后台提供的新版全局网站代码。

---

### 总结配置示例

假设你获取到的百度 ID 是 `abcd123456`，Google 的 GA4 ID 是 `G-XYZ123`，你的配置文件应该修改为：

```yaml
# Analytics settings
# Baidu Analytics
ba_track_id: abcd123456   # 替换为你从百度获取的32位字符

# Google Analytics
ga_track_id: 'G-XYZ123'   # 建议使用引号包裹，替换为你的 G- ID
ga_domain: auto

```
